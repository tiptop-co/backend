### UX

#### Главная страница

Реализует действия:
1. Вызвать официанта

Посетитель попадает на главную страницу после сканирования QR-кода.
На странице расположены:
1. Номер стола
2. Кнопка перехода в меню
3. Кнопка перехода в корзину
4. Кнопка перехода в меню оплаты
5. Кнопка `Вызвать официанта`

#### Меню

Реализует действия:
1. Просмотреть информацию о блюдах в меню
2. Отфильтровать блюда по категории (Горячее, десерт и т.д.)
3. Выбрать блюда из меню и добавить их в корзину

Меню - список карточек блюд.
Карточка содержит:
1. Название
2. Фотография
3. Стоимость

При нажатии на карточку раскрывается подробное описани блюда, включающее:
1. Текстовое описание
2. Состав
3. КБЖУ

- По умолчанию отображаются карточки блюд всех категорий.
- Категории отфильтрованы в строго заданном заранее проядке.
- Над меню расположен селектор с категориями.
- При нажатии на категорию в меню отображатся карточки блюд только данной категории.
- Категория отображается в селекторе, если в меню данного заведения есть хотя бы одно блюдо, принадлежащее ей.

Рядом с каждой карточкой в меню расположены:
1. Кнопка "Добавить в корзину"
2. Счетчик (>0)

При нажатии на кнопку в корзину добавляется число блюд, равное значению счетчика на момент нажатия кнопки

#### Корзина

Реализует действия:
1. Просмотреть корзину
2. Удалить блюда из корзины
3. Выбрать блюда из корзины и заказать их

В корзине отображаются карточки блюд, которые еще не были заказны.
Рядом с каждой карточкой расположены:
1. Кнопка "Удалить"
    При удалении удаляются все экземпляры блюда (Например, если в корзину было добавлено 3 салата, то они удалятся все)
2. Чекбокс
3. Счетчик (>0)
    Позволяет изменить число экземпляров блюда

Внизу страницы расположены:
1. Кнопка `Заказать`
2. Текстовое поле с текстом-подсказкой `Пожелания к заказу`

При нажатии на кнопку `Заказать` формируется заказ из блюд с установленными чекбоксами и числом экземпляров из соответствующих счетчиков.

#### Меню оплаты

Реализует действия:
1. Выбрать блюда из меню оплаты и оплатить их
2. Оставить чаевые

В меню оплаты расположены карточки блюд, которые были заказаны.
Рядом с каждой карточкой расположены:
1. Чекбокс
2. Счетчик (>0)
    Позволяет изменить число экземпляров блюда

Внизу страницы расположена:
1. Фамилия и Имя официанта
2. Текстовое поле для ввода суммы чаевых 
3. Сумма для оплаты (Сумма выбранных для оплаты блюд + сумма чаевых)
4. Кнопка `Оплатить`
